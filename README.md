## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/meletnaza-prog/figma-rr-chat.git
   cd figma-rr-chat
   ```

2. **Open the application**:
   - Simply open `index.html` in your web browser
   - Or use a local server for development:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js
     npx serve .
     ```

3. **Start chatting**:
   - Type your message in the input field
   - Press Enter to send or Shift+Enter for new line
   - Use voice input by clicking the microphone button
   - Attach files using the paperclip button

## Features in Detail

### Chat Interface
- Clean message bubbles with different styles for user and AI messages
- Timestamp display for each message
- Smooth animations for new messages
- Auto-scroll to latest messages

### Voice Input
- Browser-based speech recognition
- Visual feedback during recording
- Automatic text insertion into input field

### File Attachments
- Support for multiple file types (images, PDFs, documents)
- Visual file attachment indicators
- AI response to attached files

### Chat Management
- Multiple chat sessions
- Chat history persistence
- Clear history functionality
- Export/import chat data

### Responsive Design
- Mobile-first approach
- Adaptive layout for different screen sizes
- Touch-friendly interface elements

## Browser Support

- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge

**Note**: Voice input requires a modern browser with Web Speech API support.
